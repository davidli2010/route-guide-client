# route_guide_client

A gRPC example using [tonic](https://github.com/hyperium/tonic), comes from [gRPC tutorials](https://grpc.io/docs/tutorials/basic/java/),
but only client.

[Here](https://github.com/davidli2010/tonic-route-guide) is a complete implementation including server side.

## Run

You can inspect this example by compiling and running the example client in one shell session:
```Bash
cargo run
    Finished dev [unoptimized + debuginfo] target(s) in 0.13s
     Running `target/debug/client 59519`
...
```
